# my dotfiles

my dotfiles

## Instructions for me

0. clone repo to ~/
0. `cat source ~/dotfiles/vim/vimrc >> ~/.zshrc`
0. `ln -s ~/dotfiles/vim/vimrc ~/.vimrc`
0. `ln -s ~/dotfiles/vim ~/.vim`
